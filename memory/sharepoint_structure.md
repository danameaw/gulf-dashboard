## SharePoint (Team Site) — WTE Projects
**Site:** `https://gulfgroup.sharepoint.com/sites/eng/Private Documents/00 Project Reports/`
**Structure:** `2026/{WW}_{YYMMDD}/{ProjectType}/`
- Example: `2026/24_260617/iWTE/` → WTE weekly reports

## LNG Report — Data Location (Section 3. Progress)
**Section 3.1.1 Overall Progress Table**
| Column | Meaning |
|--------|---------|
| Package | LNG Tank / BOP / Marine / Commissioning |
| Engineering Plan/Actual | Weight 5% |
| Procurement Plan/Actual | Weight 45% |
| Construction Plan/Actual | Weight 45% |
| Commissioning Plan/Actual | Weight 5% |
| Subtotal Plan | Overall cumulative plan % |
| Subtotal Actual | Overall cumulative actual % |
| Variance | Actual - Plan (highlighted green/red) |

**Overall Progress row** = use for dashboard Plan% and Actual%

## Week Numbering
Folder name format: `{WW}_{YYMMDD}` where WW = week number, YYMMDD = date of report
- Week 20 = `20_260520`
- Week 21 = `21_260527`
- Week 22 = `22_260603`
- Week 23 = `23_260610`
- Week 24 = `24_260617`
